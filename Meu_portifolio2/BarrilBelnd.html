Hud de um jogo de faroeste

Eduardo Skalecki Cardon

Ano de criação: 2021

Estilo: 3D

Gênero: Faroeste

Para esse projeto foi utilizado o software de modelagem Blender para criar formas e deformá-las a fim de criar o objeto desejado.

Esse projeto foi realizado como treino para um projeto de uma casa de campo medieval.
