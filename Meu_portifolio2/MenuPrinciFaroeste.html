Menu principal de um jogo de faroeste

Eduardo Skalecki Cardon

Ano de criação: 2021

Estilo: 2D

Gênero: Faroeste

Para esse projeto foi utilizado o software Inkscape, desse modo os desenhos foram feitos utilizando formas, como triângulos, retângulos e círculos. Além disso utilizei algumas fontes personalizadas

Esse projeto foi realizado para um possível futuro jogo de faroeste
