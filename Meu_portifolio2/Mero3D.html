Ambiente virtual Mero 3D

Eduardo Skalecki Cardon e William Fabiano da Silva Filho

Ano de criação: 2022

Estilo: 3D

Gênero: Educativo

Para esse projeto foi utilizado a Engine Unity para programação e montagem do ambiente, para a programação utilizamos o programa Visual Studio Code, além desses dois softwares utilizamos também o Blender para modifarmos modelagens

Esse projeto foi feito pelo Instituto Federal do Paraná em parceria com o Projeto Meros do Brasil para conscitizar sobre o risco de extinção do peixe Mero
