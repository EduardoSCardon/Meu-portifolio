Modelagem de uma casa de campo 3d

Eduardo Skalecki Cardon

Ano de criação: 2021

Estilo: 3D

Gênero: Medieval

Para esse projeto foi utilizado o software de modelagem Blender para criar formas e deformá-las a fim de criar o objeto desejado.

Esse projeto foi realizado para um trabalho que tinha como objetivo a modelagem de um ambiente medieval. Como é possível ver há diversas foras e objeto, esse foi um ótimo projeto para prática e aprendizado pois foram necessárias várias ferramentas do Blender
